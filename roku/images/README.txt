Global Pulse - image assets
===========================

This folder intentionally ships WITHOUT binary image files, because the
project was generated by a text-only build process that cannot emit PNG/JPG.
The channel still installs and runs without them. Here is what each asset is
for and how to add it.

--------------------------------------------------------------------
1) Channel icon + splash (optional, polish)
--------------------------------------------------------------------
Referenced (commented out) in ../manifest. Drop these files here and
uncomment the matching lines in the manifest:

  icon_focus_hd.png    290 x 218   channel poster (HD)
  icon_focus_fhd.png   336 x 210   channel poster (FHD)
  icon_side_hd.png     108 x 69    small grid icon
  splash_hd.jpg        1280 x 720  boot splash (HD)
  splash_fhd.jpg       1920 x 1080 boot splash (FHD)

If these are missing, Roku uses a default placeholder icon. Not required
for sideloading/testing.

--------------------------------------------------------------------
2) The world map  (loaded from the network, not bundled)
--------------------------------------------------------------------
The equirectangular base map is loaded by URL in ScreensaverScene.brs
(mapPoster.uri), so no local file is needed. Default:

  https://upload.wikimedia.org/wikipedia/commons/c/cb/Blue_Marble_2002.png

It is a public-domain NASA Blue Marble equirectangular (2:1) image that
Wikimedia allows hotlinking. Alternates if that ever moves:

  https://upload.wikimedia.org/wikipedia/commons/8/83/Equirectangular_projection_SW.jpg
  https://upload.wikimedia.org/wikipedia/commons/e/ec/World_map_blank_without_borders.svg (avoid SVG on Roku)

To bundle the map instead of hotlinking (more reliable, recommended for a
shipping build): download a 2:1 JPG to images/worldmap.jpg and change the
line in ScreensaverScene.brs init() to:
  m.mapPoster.uri = "pkg:/images/worldmap.jpg"

--------------------------------------------------------------------
3) Marker glow
--------------------------------------------------------------------
The glowing event dots are drawn with stacked translucent Rectangle nodes
(no image needed). For a softer, rounder glow, add a radial-gradient PNG
here (e.g. glow.png, 64x64, transparent background, white soft dot) and in
ScreensaverScene.brs buildMarkers() replace the "glow" Rectangle with a
Poster:
  glow = marker.createChild("Poster")
  glow.uri = "pkg:/images/glow.png"
  glow.blendColor = col          ' tints the white dot to the category color
  glow.width = 40 : glow.height = 40 : glow.translation = [-20,-20]

--------------------------------------------------------------------
4) QR code
--------------------------------------------------------------------
Generated on demand from a public QR service (api.qrserver.com); no local
asset needed. See ScreensaverScene.brs showQR().
